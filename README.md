## 🎬 Key Features & Highlights

This portfolio isn't just a collection of links; it's a "production terminal" highlighting scalable architecture, high-velocity engineering, and narrative design.

- **"Director's Cut" Mode**: An interactive viewing mode that overlays a live, cinematic terminal script showcasing my thought processes and "system logic" in real-time.
- **Magnetic Micro-Animations**: Buttons, links, and cards exhibit subtle, physics-based "magnetic" snapping when hovered over, powered by advanced `framer-motion` hooks (`useSpring`, `useMotionValue`).
- **AJAX Signal Hub**: A fully functional, serverless contact form relaying direct emails via the FormSubmit API, avoiding page redirects and maintaining the Single Page Application (SPA) illusion.
- **The Forge (Projects Area)**: A beautiful, scalable grid system showcasing significant technical milestones ("Data Insights Studio", "Translator Pro") and an interactive metric tracking expanded open-source repositories.
- **Responsive "Cinematic" Aesthetic**: Deep emeralds, sleek blacks, glass-morphism overlays, and smooth parallax effects, optimized perfectly across mobile, tablet, and high-DPI desktop viewports.

## 💻 Technical Architecture (Tech Stack)

This project leverages modern React primitives and utility classes to ensure lightning-fast performance while rendering heavy animations.

*   **Framework:** [Next.js (App Router)](https://nextjs.org/)
*   **Language:** TypeScript / TSX
*   **Styling Engine:** [Tailwind CSS](https://tailwindcss.com/)
*   **Animation Physics:** [Framer Motion](https://www.framer.com/motion/)
*   **Iconography:** [Lucide React](https://lucide.dev/)
*   **Form Handling:** [FormSubmit.co API](https://formsubmit.co/)
*   **Deployment & Hosting:** [Vercel](https://vercel.com)

## 🚀 Quick Start (Local Development)

To run this studio experience locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Bhanunar1/portfolio.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Boot the development terminal (server):**
   ```bash
   npm run dev
   ```

5. **Open localhost:**
   Navigate to `http://localhost:3000` in your web browser.
